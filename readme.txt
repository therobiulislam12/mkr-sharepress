A simple and lightweight plugin that adds customizable social media share buttons with popup functionality.

== Description ==
Social Share Press Plugin allows you to add share buttons to your posts on WordPress site. It supports popular social media platforms such as Facebook, Twitter, LinkedIn, WhatsApp, Telegram, Reddit, Pinterest, Skype, and more, with share buttons that open in popups for a seamless user experience.


== Installation ==

1. Upload <strong>social-share-press</strong> to the /wp-content/plugins/ directory, or search and install it from WP directory.
2. Activate the plugin through the Plugins menu in WordPress.
3. Now select from the Admin menu SSP Settings and edit your settings.

== Frequently Asked Questions ==

= How can I add or remove social media platforms? 
= Use the $permitted_links array to include only the social networks you want.

= Can I change the appearance of the icons? 
= Yes, the plugin supports Font Awesome icons, so you can customize the CSS classes to style them as you wish.

= Does this plugin track any user data? 
= No, this plugin does not track or store any user data.
